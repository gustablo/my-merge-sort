# Merge sort Algorithm 
    implemented merge sort algorithm

## Run

```bash
$ node index.js
```
## How to use
in your js file:

```javascript
const mergesort = require('./merge-sort');

const unsortedArr = [5, 1, 2, 6, 7, 8, 3, 4];

console.log(mergesort(unsortedArr)); // expect: [1, 2, 3, 4, 5, 6, 7, 8]
```
